### 🚀 Avançado
- Tratamento de exceções
- Generics, Set, Map
- Programação funcional
- Expressões Lambda
- JDBC

### 🌐 Desenvolvimento
- Web services com Spring Boot e JPA
- Interface gráfica com JavaFX
